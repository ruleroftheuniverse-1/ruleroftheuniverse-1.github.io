# ruleroftheuniverse-1.github.io

Nichole Smith's public routing hub for research, writing, projects, CV, archive, and contact information.

## v0.1 structure

The canonical public site is a set of standalone HTML files in the repository root. It uses one root-level stylesheet, requires no framework or build step, and is served directly by GitHub Pages. `.nojekyll` disables Jekyll processing.

The `_layouts`, `_includes`, `_pages`, `_config.yml`, `cv.md`, and `NICHIdb` paths are retained as archived or development material; they are not required to render the public site.
